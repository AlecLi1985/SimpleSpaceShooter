Controls
---------------------
Keyboard/Mouse		Gamepad
W, S 			South/East button	= 	Throttle Up/Down
A, D 			Left/Right trigger	= 	Roll Left/Right
Mouse 			Left Stick		= 	Yaw/Pitch
Left Mouse Button 	West button		= 	Fire
Escape 			Start			= 	Pause
