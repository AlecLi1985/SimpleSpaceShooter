Controls
---------------------
Keyboard/Mouse		
- W, S 			= 	Throttle Up/Down	
- A, D 			= 	Roll Left/Right
- Mouse 			= 	Yaw/Pitch
- Left Mouse Button 	= 	Fire
- Escape 			= 	Pause

Gamepad
- South/East button	= 	Throttle Up/Down
- Left/Right trigger	= 	Roll Left/Right
- Left Stick		= 	Yaw/Pitch
- West button		= 	Fire
- Start			= 	Pause
